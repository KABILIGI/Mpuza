 
<h1>Mpuza Social Networking Platform</h1>

This project is a high-fidelity front-end implementation of the Mpuza interface, a professional social networking and job recruitment platform. It features a multi-column layout designed for user engagement, job tracking, and professional networking.

## Core Features
Custom Professional Header: A #e53935 themed navigation bar containing brand identity, global search with a "Q" placeholder, and essential navigation links (Home, My Network, Courses, Notifications, and Job Alerts).

Three-Column Responsive Layout:

Left Sidebar: Displays the user profile (e.g., kalisa Felix), professional statistics (profile viewers, job applications, recruiter interviews), and company management tools for brands like Airtel.

Central Feed: Includes a post creation area with audience selection and job posting features (supporting a specific US$ 99.99 price point), a horizontal "Urgent Hiring" carousel, and detailed job feed items with real-time countdown timers.

Right Sidebar: Features a "Favourited Jobs" quick-access module and motivational community banners.

Job Management: Detailed job descriptions for companies like Airtel Rwanda, including vacancy counts, experience requirements, and specific educational qualifications.

Engagement Tools: Functionality for users to follow companies, like, comment, share, and favorite job offers.

## Technical Specifications
HTML5 & CSS3: Utilizes a semantic structure and modern CSS Flexbox/Grid for the complex three-column layout.

Iconography: Powered by Font Awesome for professional-grade icons across the navigation and sidebar modules.

Typography: Clean sans-serif font stacks (Arial/Helvetica/System-UI) to ensure readability and a modern professional feel.

Color Palette:

Primary Header: #e53935 (Light Sky Blue)

Action Blue: #004182 or #34aadc

Notification Red: #004182

Accent Pink: #e91e63

## Project Structure
index.html: Contains the structure for the header, sidebars, and main content feed.

style.css: Contains all layout positioning, custom color variables, and component-specific styling (cards, badges, and buttons).

Assets: Placeholders for user avatars, company logos (Airtel, One Acre Fund), and promotional banners.
